# Globo Personalizada — UX Design

Este projeto propõe uma experiência digital adaptativa para o ecossistema Globo, baseada em personalização por contexto e acessibilidade integrada.

A proposta considera o comportamento do usuário ao longo do dia, adaptando a interface para diferentes momentos, como manhã e noite, além de incluir uma versão acessível voltada para pessoas com daltonismo.

## Principais conceitos

- personalização por contexto  
- experiência adaptativa  
- acessibilidade digital  
- redução da dependência de cores  
- uso de ícones e contraste
